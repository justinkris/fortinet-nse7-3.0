LoFinance is a small (fictional) financial-services company famous for freezing customers' money in an ice cube — at least on the logo. It has a Head Office (HQ) in Wollongong and two branches (Branch A and Branch B). You are the intermediate network engineer responsible for building its entire security infrastructure from scratch, inside a single Proxmox host.

Every lab follows the same rhythm:

  1. Objective — what you will be able to do afterwards.
  2. Why this matters — the real operational problem this technology solves.
  3. Think First — Socratic questions. Write your predictions before reading on. This is the single most important habit in the whole guide: prediction before configuration is how you build a mental model instead of a muscle memory.
  4. Mental Model — an analogy that will survive in your head long after the CLI syntax fades.
  5. Configuration — CLI and GUI, in that order of importance for the NSE7 exam.
  6. Verification — proving it works, with expected output.
  7. Reflection — reasoning questions, not recall questions.
  8. Challenge — a variation that forces you to adapt, not repeat.
Study discipline

Never copy-paste a command you cannot explain. If a command surprises you, stop and run its ? help, or find the concept section it belongs to. The exam tests output interpretation far more than configuration syntax.

Versions

This guide targets FortiOS 7.6, FortiManager 7.6, and FortiAnalyzer 7.6. Menu names occasionally move between minor releases; when a GUI path differs, trust the CLI, which is far more stable.
