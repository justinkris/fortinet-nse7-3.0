Routes That Teach Themselves

TRIGGER PHRASE
"47 static routes, all typed by hand." → statics don't scale because they have zero topology awareness, not because typing is slow.
CORE MODEL
Link-state (OSPF) = every router keeps an identical LSDB, runs Dijkstra independently. Never trusts secondhand rumour like distance-vector does.
TIMING
Refresh every 30 min; triggered updates only on real change. Quiet protocol.
AREAS
ABR = full LSDB per area, exports only Type 3 summary (prefix+cost) across boundaries — discards detail on purpose.
BACKBONE RULE
All areas → Area 0.0.0.0 only. No direct area-to-area links. Why: summaries lose detail → loop detection impossible → OSPF prevents the risky topology shape instead.
ROLES STACK
Internal / Backbone / ABR / ASBR are NOT mutually exclusive. One router can be all but "internal" at once.
