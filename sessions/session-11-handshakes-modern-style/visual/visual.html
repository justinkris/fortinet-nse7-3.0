🎨 Session 11 Visual — The Four Envelopes

A Packet Journey through the modern handshake, from LoFinance HQ to Parramatta.

Recommended visual style: Packet Journey. This session's core idea is inherently sequential — three jobs, a strict order, four envelopes crossing a hostile wire. A journey format makes the ordering rule (DH before identity) something you can see, and makes aggressive mode's leak visible as a letter with no envelope at all. Five placeholders below; each contains a full generation prompt ready to copy.
1 · The Three Jobs Whiteboard
Aspect ratio 3:2 · Placement: top of page, before the journey begins

Flat minimalistic but highly detailed vector illustration, modern storybook style, cream off-white background (#faf5e9), clean bold soft-brown outlines, coral accents, muted gold highlights, sage green touches, no gradients, no photorealism. Scene: a large whiteboard in a tidy Sydney trading-firm office. On the whiteboard, three numbered items written in confident marker handwriting: "1. AGREE THE LANGUAGE" beside a small icon of two speech bubbles containing matching padlock symbols; "2. GROW THE SECRET" beside an icon of a single golden key growing like a plant from two watering cans on opposite sides; "3. PROVE WHO YOU ARE" beside an icon of an ID card inside a sealed envelope. A hand-drawn arrow loops from item 2 down and around to item 3 with the annotation "ONLY AFTER!" underlined twice. Why this image matters
It fixes the session's foundation — the job list and the ordering rule — as one picture, so every later frame is judged against this board.
Accessibility description
A whiteboard lists three handshake jobs with icons; an arrow insists identity comes only after the secret. A senior engineer in a sombrero stands beside it.
2 · The Secret Is Grown, Not Posted
Aspect ratio 16:9 · Placement: after the whiteboard, before the mode comparison
[ Image placeholder — Diffie-Hellman garden ]
Image generation prompt
Flat detailed vector illustration, cream background, soft brown outlines, coral and muted gold accents, sage greens, no gradients, storybook-meets-technical-diagram aesthetic. Composition: wide 16:9, two stylised office buildings at far left and far right — left labelled "LoFinance HQ" with the frozen-money ice-cube logo, right labelled "Parramatta Branch". Between them, a long horizontal wire drawn as a country road, with a suspicious trench-coated eavesdropper figure standing mid-road holding an ear trumpet, clearly seeing everything on the road. From each building, a gardener figure sends a single labelled seed packet down the road toward the other: packets labelled "PUBLIC HALF" in friendly lettering — these are visibly harmless, the eavesdropper shrugs at them. Inside each building, behind glass, a private locked pot labelled "PRIVATE VALUE" that never leaves. At both ends simultaneously, an identical golden key-shaped plant blooms from combining the received public seed with the local private pot; both keys glow softly and are drawn identical. Caption space at bottom for the line "The secret is grown, not posted." Do NOT show any key travelling on the road. No gradients, no photorealism, no dense math notation. Learning purpose: cement that the DH shared secret never crosses the wire — it is computed independently at both ends from private + public halves.
Why this image matters
Diffie-Hellman is the concept most likely to blur under exam pressure; a picture where the key visibly never touches the road makes the eavesdropper's failure intuitive.
Accessibility description
Two buildings exchange only public seed packets along a watched road; identical golden keys grow inside each building from private pots that never leave.
3 · Three Handshakes, One Street (mode comparison)
Aspect ratio 16:9 · Placement: centre of page — the core comparison
[ Image placeholder — Aggressive vs Main vs IKEv2 ]
Image generation prompt
Flat detailed vector illustration, cream background, bold clean line work, coral/gold/sage palette, no gradients. Composition: three horizontal lanes stacked like a triple road between the same two endpoints (HQ building left, Parramatta building right), each lane clearly titled on the left margin. TOP LANE, "IKEv1 AGGRESSIVE — 6 letters": six envelopes travel the road, but the first envelopes are torn open with an ID card sticking out for all to see; a megaphone icon on the first envelope literally shouts a name-tag reading "I AM HQ!" into the street; the trench-coated eavesdropper gleefully writes it down. Amber warning tint on this lane. MIDDLE LANE, "IKEv1 MAIN MODE — 9 letters": nine sealed envelopes queue bumper-to-bumper like slow traffic, a small snail icon at lane's end; envelopes carrying ID cards are properly sealed with wax. BOTTOM LANE, "IKEv2 — 4 letters": exactly four envelopes — the first two are open postcards (safe contents: a menu card and a seed packet), the last two are luxurious sealed gold-wax envelopes containing ID cards; a subtle speed-line effect shows this lane is fastest AND the eavesdropper, arms crossed, sees nothing useful. Small scoreboard chip at right edge: 6 / 9 / 4. Do NOT include dense protocol payload text or vendor logos. Learning purpose: one glance shows aggressive = fast but shouting, main = safe but slow, IKEv2 = both properties — the false dilemma refused.
Why this image matters
The exam's favourite discriminators — message counts and ID exposure — live in this single frame, tied to the story's "shouts your name" line.
Accessibility description
Three lanes compare handshakes: aggressive mode's six letters include an ID shouted openly; main mode's nine sealed letters crawl; IKEv2's four letters are fast, with identities sealed in the final two.
4 · The Four Envelopes (IKEv2 exchange anatomy)
Aspect ratio 16:9 · Placement: immediately after the comparison
[ Image placeholder — IKE_SA_INIT + IKE_AUTH journey ]
Image generation prompt
Flat detailed vector illustration, cream background, coral/gold/sage palette, clean bold outlines, no gradients. Composition: a left-to-right packet journey in four numbered stops along a single wire-road between HQ (left) and Parramatta (right). Stop 1, labelled "IKE_SA_INIT →": an open postcard travelling right carrying two small items — a restaurant-style menu titled "CIPHERS" and a seed packet labelled "DH PUBLIC". Stop 2, "← IKE_SA_INIT": the return postcard carrying a menu with one dish circled and Parramatta's own seed packet. Between stops 2 and 3, a vertical dotted line across the whole scene labelled "THE SECRET NOW EXISTS" — on its left the road is bare; on its right the road is wrapped inside a translucent sage-green tube (the protected channel). Stop 3, "IKE_AUTH →": a sealed gold-wax envelope inside the tube containing an ID card and a small handshake token, plus a tiny piggy-backed parcel labelled "FIRST CHILD SA" riding on top of the envelope. Stop 4, "← IKE_AUTH": the mirrored sealed reply. At far right, the two buildings connected by a completed sturdy tunnel with a small "4 MESSAGES" flag planted on it. Do NOT include quick-mode imagery, extra messages, or dense hex payloads. Learning purpose: make the two-exchange structure, the plaintext/encrypted boundary, and the piggy-backed child SA visually inseparable from the count of four.
Why this image matters
This is the frame the student personally derived in-session; the image should let him re-derive it at a glance forever.
Accessibility description
Four messages between two buildings: two open postcards (proposals and DH publics), then a dotted line marking where the secret is born, then two sealed envelopes carrying identities with a small child-SA parcel piggybacked.
5 · The Nightclub With Two Doors (network-id)
Aspect ratio 4:3 · Placement: end of page
[ Image placeholder — network-id nightclub ]
Image generation prompt
Flat detailed vector illustration, cream background, coral/gold/sage palette, warm evening-lit but flat-shaded, no gradients. Scene: a stylish small nightclub facade named "CLUB HQ" with the LoFinance frozen-money ice-cube logo as its neon-style flat sign, and — crucially — TWO absolutely identical front doors side by side at the SAME street address plaque reading "203.0.113.10". Both doors have identical bouncers (drawn as friendly firewall-brick figures with sunglasses). Two guests approach from the same taxi labelled "PARRAMATTA": each holds up a ticket; the tickets are identical in shape and colour EXCEPT for the event line — one reads "EVENT: NETWORK-ID 1 · ISP1 SALSA NIGHT", the other "EVENT: NETWORK-ID 2 · ISP2 JAZZ NIGHT". Each bouncer checks the event line and waves the matching guest to the correct door; subtle matching colour ribbons (coral for 1, sage for 2) run from each ticket to its door. A small chalkboard beside the doors reads "IKEv2 GUESTS ONLY — Fortinet house rules". In the background window, Junior watches with dawning comprehension while Cipher, arms folded, gives a single approving nod. Do NOT show different street addresses, different door shapes, or any IKEv1 signage. Learning purpose: same gateway pair (one address), multiple tunnels (two doors), disambiguated only by the tag inside the negotiation (the event on the ticket) — Fortinet-proprietary, IKEv2-only.
Why this image matters
network-id is a small fact with a big exam footprint; binding it to the co-owned nightclub analogy makes "same gateway pair" scenarios instantly recognisable.
Accessibility description
A club with one street address but two identical doors; two guests are sorted by the event name printed on otherwise identical tickets, labelled network-id 1 and 2.
