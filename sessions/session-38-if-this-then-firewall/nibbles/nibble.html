The one rule

Push at config time, not trigger time. Stitches authored on the root, pushed to leaves; leaves execute local copies — root down ≠ automation down.

Stitch grammar

Trigger + action(s). No verb-less sentences — even backups have a trigger: the Schedule trigger (once/hourly/daily/weekly/monthly, root's clock).

High-CPU pipeline

cpu-use-threshold default 90. Diagnostics script → Sequential + 30s delay → email body = %%results%%. Let the kettle boil before pouring.

The 3am chain

FC logs → FAZ finds IoC → FGT verifies connected → EMS orders → endpoint arrests ITSELF. Four required: FGT·FAZ·EMS·FortiClient. Not on Linux FC.

Why the endpoint

"You can only block what is inline." Same-L2 traffic never meets a firewall; quarantine travels with the laptop and blocks the device, not a DHCP lease.

Triptych
