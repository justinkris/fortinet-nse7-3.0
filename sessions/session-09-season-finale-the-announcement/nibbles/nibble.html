Season Finale — 1-Minute Refresher

Trigger words → mental model
Role name (NGFW/ISFW/DCFW/DEFW) → position + traffic path, not hardware.
VLAN "segmentation" → gateway must be on the firewall, not a switch.
802.1Q → egress adds tag, ingress removes it.
A VDOM doesn't move a building.
Inter-VDOM link → route + policy, BOTH sides, independently.
NAT vs Transparent VDOM → routed hop vs invisible L2 bridge.
VLAN reach = VDOM reach → both are chassis/trunk-bound. Logic ≠ geography.
