One Box, Many Firewalls

Session 06 · 1-min nibble
VDOM = VRF, one layer up. One box, up to 10 isolated firewalls, shared hardware.
Global VDOM ≠ a firewall. Device-wide settings only. No Policy/VPN/User&Auth.
Admin VDOM = zero traffic, max 1 per box, mgmt/updates/licensing only.
NAT mode = full L3 firewall. Transparent = invisible L2, still needs 1 manageip.
LAN Extension = Network + VPN only. Tunnels a remote LAN via IPsec/VXLAN.
10-VDOM limit is default, not fixed — license to extend, don't buy a box first.
