Crying Wolf

Tuning IPS = managing two opposite failures at once.
Cry wolf

False positive — legit traffic it CAN read gets blocked. All health checks green; truth in the IPS log. Fix: Signature + Monitor, above the Filter.

Miss wolf

False negative — exploit it CAN'T read (inside HTTPS / encrypted DNS) sails through. Fix: full inspection.

Depth

Certificate = envelope. Full = letter. IPS reads the letter → inside encryption always needs full.

Mode

Flow = stream (sliding buffer). Proxy = reassemble whole file (AV needs this). Full inspection runs in both.

Rule

Encryption decides depth. Reassembly decides mode. HTTP → cert; HTTPS → full.

One line to recall it all
Certificate reads the envelope; full reads the letter. Flow matches a pattern mid-stream — no whole file. AV needs the file, so AV means proxy.
