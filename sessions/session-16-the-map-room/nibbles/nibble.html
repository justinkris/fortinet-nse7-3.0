The Map Room

The Pipeline

LSDB → Dijkstra → tree → routes → routing table. Routing table = last exhaust of the pipe. Rumour spreads answers; link-state spreads facts.

LSA

Advertises links, not routes — a router's firsthand facts (connected nets + costs). LSDB = all LSAs collected. Refresh every 30 minutes; changes → triggered update.

Cost

16-bit, 1–65,535, lowest total wins. Sum of exit costs. Directional → asymmetry breaks stateful firewalls. Symmetry is your job, not OSPF's.

Map vs Trees

Same LSDB per area; different trees (own root each); zero contradictions. Loops = unsynchronised databases. Full = the bet is safe.

Areas

Separate LSDB per area (never merged). Smaller DBs + contained blast radius. Backbone = 0.0.0.0, mandatory hub. ID: 10 ≡ 0.0.0.10.

database brief
