SESSION 16 · STUDY BITE · ~5 MINUTES

The Map Room — OSPF Internals I

The Pipeline (know the order cold)

LSDB → Dijkstra → SPF tree → OSPF routes → routing table. The routing table is the last exhaust of the pipe. The map's payoff is independence: hear one fact, recompute everything locally. Distance-vector waits on daisy-chained rumour.

"Rumour spreads answers. Link-state spreads facts."

LSAs & the LSDB

  • A router's only firsthand knowledge: its own connected networks + interface costs. That's what an LSA advertises — links (facts), never routes (answers).
  • LSDB = all LSAs collected (own + received). LSAs are acknowledged.
  • Quietness: full re-advertisement every 30 minutes; otherwise silence unless a topology change fires a triggered update.
Exam Trap"Every 30 seconds" = RIP. OSPF = 30 minutes + triggered. Read the unit twice.

Cost

  • 16-bit, 1–65,535. Lowest total path cost wins.
  • Path cost = sum of EXIT interface costs along the way.
  • Cost is directional — each side pays its own exit price. Asymmetric costs on parallel paths → forward and return traffic take different paths → stateful firewalls drop the return (SYN one way, SYN-ACK the other, no session state). Every routing table "technically correct."
"Cost is paid on exit. Symmetry is your job, not OSPF's."

Same Map, Different Trees

  • Identical: the LSDB (per area). Different: each router's tree (rooted at itself).
  • No contradictions because all trees derive from the same facts. Loops in link-state = unsynchronised LSDBs.
  • Full = the two neighbours' LSDBs are identical & synchronised = routing through each other is now a safe bet.

Areas

  • Flat LSDB's two costs: unbounded size + blast radius (one flap anywhere = SPF everywhere).
  • Area = 32-bit ID, decimal (10) ≡ dotted (0.0.0.10). Separate LSDB per area; multi-area router keeps one FULL LSDB per area — never merged.
  • Benefits: smaller DBs, contained blast radius, summarisation at borders. Cost: design + troubleshooting complexity.
  • Backbone = Area 0.0.0.0, mandatory; all areas connect to it.

  • AS External section: holding external LSAs ≠ being the ASBR — check ADV Router. Advertising them is what makes it you.
