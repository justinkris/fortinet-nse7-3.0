SESSION 04 · NIBBLE · 60 SECONDS

The Floor Wars — Trigger Recall

No VLAN

= one broadcast domain = no possible inspection point. Ever.

Tag Rule #1

FortiGate never adds a tag on ingress — the switch already did.

Tag Rule #2

FortiGate CAN change a tag when routing (strip → route → re-tag on egress). Tag = property of the wire, not the packet.

VLAN interface

= physical interface (which wire) + vlanid (which tag). Both required.

Real segmentation

= gateway on FortiGate, not a switch. Otherwise: unlocked drawers.

First troubleshooting move

diagnose sys session list — no session = Layer 2 problem, not policy.
