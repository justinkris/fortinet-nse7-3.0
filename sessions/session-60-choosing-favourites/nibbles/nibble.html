One Minute on Choosing Favourites

Triggers only. If a line does not re-fire the model, go back to the Bite.

Sixty seconds. Eight triggers.

Not to learn from — to re-fire the mental model.

The frame
BGP is never wrong. It is badly advised.
A wrong path is usually an absence of policy, not a fault.
Route selection
The tie-breaker list is an argument in order. First difference wins.
Weight → local pref → local origin → AS path → origin → MED → IGP metric → EBGP>IBGP → ECMP(10) → router ID. Nothing below the deciding line is ever read.
Tactics
Argue above the line you are losing on.
Line 10 — lowest router ID — means someone else's addressing chose your path.
Location
Best path is a RIB-in → Local RIB event.
Candidates queue in RIB-in; only the winner enters the Local RIB. Policy runs first; the doorman reads the result.
Scope
Weight is a private opinion. Local preference is a company policy.
Who needs to agree? One box → weight (never advertised). One AS → local pref (stripped at the edge).
Categories
Ask how far it travels and the category answers itself.
COMMUNITY travels (optional transitive). MED must not (optional non-transitive). LOCAL_PREF never leaves home (well-known discretionary). ORIGIN / AS_PATH / NEXT_HOP are well-known mandatory.
Origination
Redistribution is a bucket. config network is a tap.
network-import-check: exact prefix AND mask, active, in the ROUTING TABLE. Covered ≠ matched. Disabling it is global and silent.
Filters
Every FortiOS filter ends in an implicit deny.
Route map, prefix list, access list — same family trait. One permit rule means “this and nothing else.” Only route maps can modify attributes.
Exam technique
Score first, count last.
The count is a checksum, not a stopping condition.
