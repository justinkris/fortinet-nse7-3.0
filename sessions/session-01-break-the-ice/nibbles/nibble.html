SESSION 01 · NIBBLE · 60 SECONDS

Break the Ice — Instant Recall

1
A firewall only inspects traffic that crosses it. No inspection point in the path → no logs, no detection. Check the path before the feature.
2
Implicit trust → lateral movement. Threats start inside (USB, personal devices, insiders). Zero trust: assume compromise, verify everywhere.
3
Segmentation = forced inspection. Gateway on the FortiGate, not the switch — or it's just labels. Goal: limit the blast radius. Ship compartments.
4
Envelope vs letter. Ports/IPs are envelopes; everything hides on 443. Only app-layer inspection reads the letter — and SSL inspection unseals it.
5
NGFW = superset. Traditional firewall (ACLs, port/IP, stateful) + app-layer analysis + UTM: AV (files) · IPS (traffic) · web filter · app control · ATP.
6
FortiGuard feeds everything. Config ≠ protection. One FortiOS across all FortiGates; one console: FortiManager.
"One flooded compartment doesn't sink the vessel." — the sentence Penny took to the board.
