1. QUICK SUMMARY (one page)
-------------------------------------------------------------------------
The traditional model — one firewall at the perimeter, trust everything
inside — is dead. Threats (zero-days, APTs, polymorphic malware, insider
threats) routinely start INSIDE the network via USB sticks, compromised
personal devices, or phished users. Because a firewall only inspects
traffic that crosses it, a perimeter-only design leaves internal flows
invisible: implicit trust enables LATERAL MOVEMENT. The answer is a
ZERO-TRUST posture: assume compromise, verify everything, and SEGMENT the
network so inter-segment traffic is forced through inspection points
(gateways on the firewall, not the switch), limiting the BLAST RADIUS.

Port/IP filtering reads only headers ("envelopes"). With ~everything on
TCP 443, envelopes are near-worthless; you must read the "letter" — the
application layer and the content. The NGFW does this: it is a SUPERSET
of the traditional firewall (keeps ACLs, port/IP filtering, stateful
inspection) and ADDS application-layer analysis plus UTM security
engines: antivirus (files), IPS (traffic exploits), web filtering,
application control, and advanced threat protection. SSL inspection is
the ENABLER that lets those engines see inside encrypted traffic.

All engines depend on current intelligence from FORTIGUARD LABS — AI/ML
threat intel pushed globally in near real time. Config ≠ protection.

The Fortinet enterprise solution spans FortiGate 30-90 (branch), 100-900
(campus), 1000-7000 (data center), plus FortiGate VM / FortiCNF /
FortiSASE in cloud — all running the SAME FortiOS, all manageable from
ONE console: FortiManager. Where each box goes = Session 2.

DESIGN RULES
  - Controls proportional to asset sensitivity (servers first at LoFinance).
  - A VLAN whose gateway is on a switch is NOT segmentation.
  - Every inspection layer costs latency — size the box for the load.
  - No inspection point in the path = no logs, no detection. Check the
    PATH before blaming the FEATURE.

3. FLASHCARDS
-------------------------------------------------------------------------
Q: Why can't a perimeter firewall detect lateral movement?
A: It only inspects traffic that traverses it; internal host-to-host
   flows never cross it — no inspection point, no visibility, no logs.

Q: What makes network segmentation real rather than cosmetic?
A: Inter-segment traffic must be forced through an inspection point —
   the segment gateway lives on the firewall, not the switch.

Q: Define an NGFW in one sentence.
A: A firewall that incorporates traditional capabilities (ACLs, port/IP
   filtering, stateful inspection) and adds application-layer analysis
   plus UTM security features.

Q: Name the five core UTM engines.
A: Antivirus, IPS, web filtering, application control, advanced threat
   protection (sandboxing).

Q: AV vs IPS — the one-line split?
A: AV inspects files for malicious payloads; IPS inspects traffic for
   exploits and protocol anomalies.

Q: Why is SSL inspection called the "enabler"?
A: 90%+ of traffic is encrypted; without decryption every other engine
   inspects a locked box.

Q: Why do granular port ACLs fail as application control?
A: Nearly all apps ride TCP 443; ports identify envelopes, not apps —
   only app-layer inspection identifies the application itself.

Q: Where does a FortiGate learn about a brand-new threat seen elsewhere?
A: FortiGuard Labs — AI/ML global threat intelligence pushed to
   subscribed devices in near real time.

Q: Two unifiers across the whole FortiGate range?
A: One operating system (FortiOS) and one management console
   (FortiManager).

Q: Attack enters via USB, spreads internally, firewall logs show nothing.
   Root cause category?
A: Architecture, not features — the traffic never crossed the firewall.

Q: What threats killed the perimeter-only model? (name 4)
A: Zero-day attacks, APTs, polymorphic malware, insider threats.

Q: LoFinance rule: how much security does each asset get?
A: Proportional to sensitivity — and never so much inspection latency
   that the traders feel it.
