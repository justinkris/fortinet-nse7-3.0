Singapore Calling — BGP Fundamentals

The Core Shift

OSPF shares a whole topology map between trusted routers — one truth, computed identically. BGP shares only a list of destinations and the AS path they crossed — routing as policy, negotiated between organisations that don't fully trust each other.

Autonomous System

A set of routers/networks under one administration, identified externally by one AS number. Internally it can run any IGP (OSPF stays untouched).

AS_PATH — The Only Map BGP Has

Every route carries the ordered list of AS numbers it crossed. Read right-to-left: origin first, most recent hop last.

LOOP PREVENTION
If a router sees its own ASN already in the AS_PATH, it rejects the route.

Stub / Multihomed / Transit

  • Stub — one AS connection, local traffic only
  • Multihomed — multiple AS connections, still only local traffic
  • Transit — carries traffic that isn't its own (ISP example)
Trap: multihoming ≠ automatic transit. What prevents accidental transit is prefix filtering, not AS_PATH. Two different tools, two different jobs.

Private ASN Range

64512–65534 — same idea as RFC 1918 private IPs. Used for privately-agreed BGP between organisations that aren't peering with the public internet.
