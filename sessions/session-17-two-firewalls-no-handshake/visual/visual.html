Visual Learning · Session 17

Two Firewalls, No Handshake

Recommended visual style for this session A hybrid of Packet Journey and Troubleshooting Decision Tree. The session's spine is a single packet's attempted trip from Parramatta to DR through three sequential checkpoints, each one silently failing for a different reason — this maps naturally onto a journey-with-checkpoints layout, while the decision-tree format reinforces the diagnostic reasoning needed to find each failure. Four illustrations below cover: the whole journey, the area-mismatch mechanism specifically (the session's hardest concept), the routing-table "lie" concept, and a memory-anchor callback linking this session to Session 15.
