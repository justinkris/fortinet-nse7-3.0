The Journey of Truth

Recommended style: a Truth-Journey through the three-layer pipeline, plus a control-room decision scene. Each placeholder carries a full generation prompt.

Recommended visual style: Packet/Truth Journey + Decision Room

Because this session is entirely about a value moving in one of two directions across three fixed stations, a journey-through-stations metaphor anchors it better than any static diagram. The second image freezes the moment of choice — the 214-line preview — so the "pull first" instinct becomes a picture, not a rule.

Why this style

The mental model we're anchoring

Everything today reduces to one shape: config is a traveller moving between three stations — Policy Layer, Device DB, Live FortiGate. Downhill travel (push) is a bulldozer that flattens whatever it lands on. Uphill travel (pull) is a scout that quietly copies what it sees. Draw that once and the four verbs sort themselves.

The four verbs

Each verb is just a trip

Retrieve and Import climb uphill (safe). Install pushes all the way down (dangerous). Revert only moves a card into the middle drawer — it hasn't travelled to the box yet.
