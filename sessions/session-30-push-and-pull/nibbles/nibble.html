Push and Pull

The one idea that fires the right mental model.

Push overwrites. Pull observes. Pull first, look, then push.
The four verbs as directions
PULL ↑ (safe)                 PUSH ↓ (dangerous)
RETRIEVE  box → device DB      INSTALL   policy → DB → box
IMPORT    DB → policy layer    REVERT    old revision → DB only
          (the bridge)                   (needs INSTALL to go live)
⚠️
The trap
Install is a BLIND OVERWRITE — an undocumented box edit is destroyed, not merged. A big install preview is a confession of drift, not a fault.
🌐
Global packages
Author once in the global ADOM → ASSIGN (flags local pkg Modified) → INSTALL. Auto-install = one move, no preview.

Revert stages · Install delivers · Import reconciles
