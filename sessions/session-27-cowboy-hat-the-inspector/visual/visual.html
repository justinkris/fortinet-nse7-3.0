Three Checkpoints, Three Lies

A packet-journey map of the sabotaged inspection chain. Recommended visual style: Packet Journey + Troubleshooting Decision Tree hybrid — each fault is a checkpoint where the inspector either lied or was silenced.

Style rationale: the packet journey
These faults are spatial and sequential — a packet moving through checkpoints that should stop it. A journey map anchors "which checkpoint owns this packet" better than a static diagram.
