Session 07 · Nibble · 1-min recall

Bridges Between Worlds

Same box → inter-VDOM link, not a cable.

Paired virtual interface, not physical ports.

Built once, both sides chosen together.

Auto-named base0 / base1.

Link + Route + Policy — on BOTH sides.

Each VDOM has its own routing table AND its own policy table.

Route first, policy second.

A route with no matching policy still drops the traffic.

No link = physically nonexistent path.

Isolation is the default. Connection is the deliberate exception.
